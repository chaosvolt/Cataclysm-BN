# アレルギー

## 基本情報

アレルギーは、遺伝性(変異による継承)への対応や、個々のアイテムに対して材料は何であるかといった困難な判断やバリエーション作成を避ける目的で、「ビタミン(栄養素)」システムをベースに構築されています。

初期の導入措置として、データの先祖返りを防ぎ、すべてのビタミンを手作業で追加する手間を省くために、材料に基づいてこれらのビタミンが適用されました。ただし、アイテムに`NUTRIENT_OVERRIDE`(栄養素の上書き設定)がない限り、調理によって何が実際に含まれたのかが正しく反映されるはずです。自然に生成されたアイテム(既製品など)には、その材料が示す通りの成分が含まれます。

もし食品に野菜アレルゲンが含まれている場合、肉食のキャラクターはそれを食べることができません。草食のキャラクターと肉アレルゲンを含む食品の関係についても同様です。

## 現在の「アレルゲン」一覧:

- `"egg_allergen"`(卵アレルゲン)
- `"fruit_allergen"`(果物アレルゲン)
- `"human_flesh_vitamin"` (人肉ビタミン：これは特殊なケースで、アレルギーではなく、食人(Cannibalism)による士気の低下や上昇の判定に使用されます)
- `"junk_allergen"`(ジャンクフード・アレルゲン)
- `"meat_allergen"`(肉アレルゲン)
- `"milk_allergen"`(ミルク・アレルゲン)
- `"nut_allergen"`(ナッツ・アレルゲン)
- `"veggy_allergen"`(野菜アレルゲン)
- `"wheat_allergen"` (小麦アレルゲン：実際にはパンや焼き菓子全般を指します)

## 現在の食事制限:

完全な禁止:

- 肉食: 野菜アレルゲン、果物アレルゲン、小麦アレルゲン、ナッツアレルゲン
- 草食 / 反芻動物(Herbivore / Ruminant): 肉アレルゲン、卵アレルゲン:
- 菜食主義者 (Vegetarian)
- アンチ植物 (Anti-plant)
- 乳糖不耐症 (Lactose intolerance)
- 果物不耐症 (Fruit intolerance)
- ジャンクフード不耐症 (Junkfood intolerance)
- 穀物不耐症 (Grain intolerance)
