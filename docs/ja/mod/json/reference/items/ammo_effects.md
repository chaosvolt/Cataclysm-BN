# 弾薬効果

ammo_effect には 2 つのタイプがあります。ハードコードされた弾薬エフェクトと JSON 化された弾薬エフェクト。

### ハードコードされた弾薬効果

| 識別子                 | 説明                                                                     |
| ---------------------- | ------------------------------------------------------------------------ |
| THROWN                 | 投げられたかどうかを確認するために使用されます                           |
| NO_PENETRATE_OBSTACLES | 障害物はできません。フェンスや強化ガラスを貫通するなど                   |
| HEAVY_HIT              | トラップを作動させ、壁に当たると大きな音を出します                       |
| NO_ITEM_DAMAGE         | 発射物が命中したアイテムやトラップはダメージを受けません                 |
| NOGIB                  | 弾薬は死体を過剰に殺したり爆発させたりしません                           |
| COOKOFF                | 火事で弾薬が爆発する                                                     |
| INCENDIARY             | 地形/ターゲットに火災を引き起こします                                    |
| NEVER_MISFIRES         | 失火しない                                                               |
| ACT_ON_RANGED_HIT      | アイテムはヒットすると使用アクションをトリガーします                     |
| APPLY_SAP              | 樹液効果をターンの間適用してダメージを軽減します                         |
| BEANBAG                | 短期間スタン効果を適用します                                             |
| BLINDS_EYES            | 目に当たるとブラインドが適用されます                                     |
| BOUNCE                 | バウンドして近くの他のターゲットに当たると、`WIDE` 効果も得られます      |
| BURST                  | 衝撃で内容物を落としますが、それも破壊されます                           |
| CUSTOM_EXPLOSION       | アイテムの爆発データを使用                                               |
| DRAW_AS_LINE           | アニメーションなしで瞬時に描画されます                                   |
| IGNITE                 | ヒットしたすべてのクリーチャーを点火する                                 |
| JET                    | ダメージが0になっても全体の効果を引く                                    |
| LARGE_BEANBAG          | BEANBAG スタン効果をより長い期間適用します                               |
| MUZZLE_SMOKE           | 発砲時に小さな煙が発生します                                             |
| NO_CRIT                | クリティカルヒットはできませんが、かすめることはできます                 |
| NO_DAMAGE              | ダメージを与えない                                                       |
| NO_DAMAGE_SCALING      | クリティカルヒット不可、かすめ不可                                       |
| NO_EMBED               | 生き物に埋め込むことはできません                                         |
| NO_OVERSHOOT           | ミスしても弾道がパスターゲットを伸ばせない                               |
| NULL_SOURCE            | 内部的に使用されており、origin は実際の生き物ではありません              |
| POISON                 | ヒットすると毒を適用し、クリティカルではさらに悪い毒を適用します         |
| PARALYZEPOISON         | ヒット時に麻痺毒を付与                                                   |
| SHATTER_SELF           | ヒットすると投げられて破壊され、内容がドロップされ、ノイズが発生します   |
| TANGLE                 | クリーチャーに縛り効果を適用すると回復可能                               |
| NET_TANGLE             | グループに結合効果を適用します。回復不可能です                           |
| WHIP                   | 攻撃を受けた野生動物を怖がらせ、リーチ攻撃でも時折武装解除を適用します   |
| BLACKPOWDER            | 有害なガンクの蓄積の原因となります                                       |
| RECYCLED               | 256 分の 1 の確率で不発になります                                        |
| RECOVER_X              | (X-1/X) の確率で、衝撃時に使用済み弾薬を 1 回分回復する                  |
| WIDE                   | 現在は効果がありません。`HARDTOSHOOT` を無効にすることをサポートします。 |
| SHOT                   | 強制的に `WIDE` 効果を適用します                                         |

### フィールド

| 識別子       | 説明                                                                                                                       |
| ------------ | -------------------------------------------------------------------------------------------------------------------------- |
| id           | (_必須_) 一意の ID。                                                                                                       |
| aoe          | (_任意_) オブジェクト。フィールドについては [aoe](#aoe) を参照してください。                                               |
| trail        | (_任意_) オブジェクト。フィールドについては [trail](#trail) を参照してください。                                           |
| explosion    | (_任意_) オブジェクト。フィールドについては [explosions](/mod/json/reference/misc/explosions/#fields) を参照してください。 |
| do_flashbang | (_オプション_) フラッシュバンエフェクトを実行します                                                                        |
| do_emp_blast | (_オプション_) Do Emp エフェクト                                                                                           |

<a id="aoe"></a>

#### 青江

ターゲットを中心とした効果

| 識別子            | 説明                                                                                      |
| ----------------- | ----------------------------------------------------------------------------------------- |
| field_type        | (_任意_) エフェクトで使用するフィールドの ID                                              |
| intensity_min     | (_オプション_) フィールドの最小強度                                                       |
| intensity_max     | (_オプション_) フィールドの最大強度                                                       |
| radius            | (_オプション_) 効果の半径                                                                 |
| radius_z          | (_オプション_) Z 軸の効果半径                                                             |
| chance            | (_任意_) 追加されたフィールドの確率                                                       |
| size              | (_任意_) AOE の大まかなサイズ                                                             |
| check_sees        | (_任意_) ソースはエフェクトの場所を参照する必要があります。エフェクトは条件の下にあります |
| check_sees_radius | (_オプション_) ソースは半径内の任意のスポットを認識する必要があります                     |
| check_passable    | (_オプション_) ターゲットは通行可能なタイル上にある必要があります                         |

<a id="trail"></a>

#### トレイル

すべての効果は発射体の経路に続く軌跡に適用されます

| 識別子        | 説明                                         |
| ------------- | -------------------------------------------- |
| field_type    | (_任意_) エフェクトで使用するフィールドの ID |
| intensity_min | (_オプション_) フィールドの最小強度          |
| intensity_max | (_オプション_) フィールドの最大強度          |
| chance        | (_任意_) 追加されたフィールドの確率          |

### 例

```json
{
  "id": "MININUKE_MOD",
  "type": "ammo_effect",
  "aoe": {
    "field_type": "fd_nuke_gas",
    "intensity_min": 3,
    "intensity_max": 3,
    "chance": 100,
    "radius": 24,
    "radius_z": 0,
    "size": 1,
    "check_passable": true,
    "check_sees": true,
    "check_sees_radius": 3
  },
  "trail": { "field_type": "fd_acid", "intensity_min": 1, "intensity_max": 3, "chance": 66 },
  "explosion": {
    "damage": 3000,
    "radius": 20,
    "fragment": {
      "impact": { "damage_type": "heat", "amount": 3000, "armor_multiplier": 2 },
      "range": 24
    }
  }
}
```
