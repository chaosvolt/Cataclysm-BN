### アイテムカテゴリ

インベントリをカテゴリ順で並べ替えた際、ここに定義されたカテゴリが表示されます。

| 識別子         | 説明                                                                                                                                                                                                                                                                                                                                  |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| id             | 固有のID。連続した1単語である必要があり、必要に応じてアンダースコア(_)を使用します。                                                                                                                                                                                                                                                  |
| name           | カテゴリの名称。インベントリを開いた際にゲーム内で表示される名前です。                                                                                                                                                                                                                                                                |
| zone           | 対応するルートゾーン(Loot Zone)。loot_zones.json を参照してください。                                                                                                                                                                                                                                                                 |
| sort_rank      | 表示時のカテゴリの並び順。値が小さいほど先に表示されます。                                                                                                                                                                                                                                                                            |
| priority_zones | 設定されている場合、条件を満たせばこのカテゴリのアイテムは「優先ゾーン」へと仕分けられます。ユーザーがゾーンマネージャーで優先ゾーンを設定していない場合は、zone プロパティで設定されたゾーンに仕分けられます。これはオブジェクトのリスト形式で、各オブジェクトは3つのプロパティ(ID: LOOT_ZONE のID、flags: フラグの配列)を持ちます。 |

```json
{
  "id": "armor",
  "name": "ARMOR",
  "zone": "LOOT_ARMOR",
  "sort_rank": -21,
  "priority_zones": [{ "id": "LOOT_FARMOR", "flags": ["RAINPROOF"] }]
}
```
