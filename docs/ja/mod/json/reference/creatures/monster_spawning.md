### モンスターグループ

#### グループの定義

| 識別子      | 説明                                                                                                                                                                                                                                                                    |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `name`      | 固有のIDです。スペースを含まない連続した単語である必要があり、必要に応じてアンダースコア（_）を使用してください。                                                                                                                                                       |
| `default`   | 既定のモンスターです。生成確率の残りの枠を自動的に埋めるために使用されます。                                                                                                                                                                                            |
| `monsters`  | 生成するモンスターを選択する際、ゲーム内では1000個のエントリが作成され、その中から1つが選ばれます。各モンスターは設定された freq（出現頻度）と同じ数のエントリを持ち、残りの枠を上記の default モンスターが埋めます。個別の定義方法については下の表を参照してください。 |
| `is_safe`   | (bool) true の場合、そのモンスターが視界に入ってもセーフモードの警告を発生させません。                                                                                                                                                                                  |
| `is_animal` | (bool) そのグループに通常の動物のみが含まれているかどうかを判定します。                                                                                                                                                                                                 |

#### モンスターの定義

| 識別子            | 説明                                                                                                                                                                                                                                                                                                                     |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `monster`         | モンスターの固有のIDです 例：`"mon_zombie"`）。                                                                                                                                                                                                                                                                          |
| `freq`            | 出現確率。1000分のxで指定します。                                                                                                                                                                                                                                                                                        |
| `cost_multiplier` | 生成するモンスターの総数に制限がある場合、この定義のモンスター1体が「何体分」としてカウントされるかを指定します。                                                                                                                                                                                                        |
| `pack_size`       | (_任意_) このグループで同時に生成されるモンスターの最小数と最大数です。(既定値: `[1,1]`)                                                                                                                                                                                                                                 |
| `conditions`      | 出現条件を制限します。有効なオプション: `SUMMER`(夏), `WINTER`(冬), `AUTUMN`(秋) `SPRING`(春), `DAY`(昼), `NIGHT`(夜), `DUSK`(夕暮れ), `DAWN`(明け方)。 時間帯や季節を複数指定した場合、それらのいずれかの条件を満たせば出現が有効になります。 (`DAY`, `NIGHT`, `DUSK`, `DAWN`) (`SUMMER`, `WINTER`, `AUTUMN`, `SPRING`) |
| `starts`          | (_任意_) 指定した時間が経過した後に、このエントリが有効になります(単位：時間)。                                                                                                                                                                                                                                          |
| `ends`            | (_任意_) (任意)指定した時間が経過した後に、このエントリが無効になります(単位：時間)。                                                                                                                                                                                                                                    |

```json
{
  "name": "GROUP_ANT",
  "default": "mon_ant",
  "monsters": [
    { "monster": "mon_ant_larva", "freq": 40, "multiplier": 0 },
    { "monster": "mon_ant_soldier", "freq": 90, "multiplier": 5 },
    { "monster": "mon_ant_queen", "freq": 0, "multiplier": 0 },
    {
      "monster": "mon_thing",
      "freq": 100,
      "multiplier": 0,
      "pack_size": [3, 5],
      "conditions": ["DUSK", "DAWN", "SUMMER"]
    }
  ]
}
```
