### モンスター派閥の定義

| 識別子         | 説明                                                                                                                                           |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| `name`         | 固有のIDです。空白を含まない連続した単語である必要があります。必要に応じてアンダースコア（_）を使用してください。                              |
| `base_faction` | 任意で設定できるベース派閥です。他派閥との関係性はこのベース派閥から継承され、他派閥がこの派閥を判定する際もベース派閥の関係性が参照されます。 |
| `by_mood`      | 「機嫌」に応じて敵対します。怒っている時は敵対し、それ以外は中立となります。他の全派閥に対するデフォルトの態度設定として機能します。           |
| `neutral`      | この派閥に対して常に中立となります。                                                                                                           |
| `friendly`     | この派閥に対して常に友好的となります。デフォルトでは、派閥は自分自身に対して友好的に設定されています。                                         |
| `hate`         | この派閥に対して常に敵対します。攻撃可能な範囲にこの派閥のモンスターがいる場合、優先的にターゲットを変更します。                               |

```json
{
  "name": "cult",
  "base_faction": "zombie",
  "by_mood": ["blob"],
  "neutral": ["nether"],
  "friendly": ["blob"],
  "hate": ["fungus"]
}
```
