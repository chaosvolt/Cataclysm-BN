# ワールドタイプ

ワールド タイプは、ディメンションに関するデータを保存するために使用されます。デフォルトのものを含む

> [!WARNING]
> 寸法はまだ実験的なものです。使用は自己責任で行ってください。

### フィールド

| 識別子               | 説明                                                                               |
| -------------------- | ---------------------------------------------------------------------------------- |
| id                   | (_必須_) 一意の ID。                                                               |
| name                 | (_必須_) ゲーム内名が表示されます。                                                |
| description          | (_オプション_) 寸法の説明                                                          |
| region_settings      | (_任意_) デフォルトは `default` で、使用する領域設定の ID です。                   |
| generate_overmap     | (_任意_) デフォルトは true、湖、森林、沼地の接続が生成される必要があります。       |
| infinite_bounds      | (_任意_) デフォルトは true、デフォルトの次元のように境界のない次元です。           |
| boundary_terrain     | (_任意_) デフォルト "boundary_terrain"、ディメンションの境界の周囲に配置する地形。 |
| save_prefix          | (_任意_) 保存するファイルの先頭に付ける文字列。                                    |
| allow_npc_travel     | (_任意_) デフォルトは false、NPC がディメンションに移動できるようにする天候。      |
| allow_vehicle_travel | (_任意_) デフォルトは false、車両がディメンションに移動できるようにする天候。      |
| scale_num            | (_任意_) この次元での移動の規模。 2 は 2 倍の長距離移動を意味します。              |
| scale_den            | (_任意_) この次元での移動の規模。 2 は移動距離が 2 倍短いことを意味します。        |
| sunrise_summer       | (_任意_) デフォルト -1 (ゲーム オプションを使用)、夏の日の出の時間                 |
| sunset_summer        | (_任意_) デフォルト -1 (ゲーム オプションを使用)、夏の日没の時間                   |
| sunset_winter        | (_任意_) デフォルト -1 (ゲーム オプションを使用)、冬の日没の時間                   |
| sunset_winter        | (_任意_) デフォルト -1 (ゲーム オプションを使用)、冬の日没の時間                   |
| sunrise_equinox      | (_任意_) デフォルト -1 (ゲーム オプションを使用)、春分点の日の出の時間             |
| sunset_equinox       | (_任意_) デフォルト -1 (ゲーム オプションを使用)、春分点の日没の時間               |
| pernament_daylight   | (_任意_) デフォルト -1 (ゲーム オプションを使用)、天気を設定します。常に日中です。 |
| pernament_night      | (_任意_) デフォルト -1 (ゲーム オプションを使用)、天候を設定します。常に夜です。   |
