# 게임 밸런스

Cataclysm: Bright Nights의 게임 밸런스는 여러 시스템 간의 균형을 유지하는 것에 관한 것입니다.

## 핵심 원칙

### 1. 현실성 우선

- 게임플레이가 허용하는 범위 내에서 현실적인 값을 목표로 함
- 현실 세계의 물리학과 논리를 따름
- 현실적인 시간 척도와 자원 가용성 사용

### 2. 진행 곡선

- 초반: 생존이 어렵고, 자원이 부족함
- 중반: 기술 개발, 안정적인 기지
- 후반: 고급 장비와 차량

### 3. 위험 대 보상

- 더 위험한 지역은 더 나은 전리품 제공
- 더 강한 적은 더 가치 있는 자원 제공
- 고급 제작은 희귀한 재료 필요

## 밸런스 고려사항

### 아이템

- **무게**: 현실적인 무게 값 사용 (kg)
- **부피**: 리터 단위의 현실적인 부피
- **가격**: 상대적 가치를 반영하는 전후 가격
- **희귀도**: 아이템의 희귀도에 따른 생성 빈도

### 전투

- **데미지**: 무기 유형과 탄약에 기반
- **방어력**: 갑옷 재질과 적용 범위에 기반
- **명중률**: 기술, 무기, 조건에 기반
- **몬스터 난이도**: 진행 단계에 적합

### 제작

- **시간**: 현실적인 제작 시간
- **재료**: 논리적인 재료 요구사항
- **도구**: 적절한 도구 요구사항
- **기술**: 복잡도에 맞는 기술 요구사항

## 밸런스 테스트

새 콘텐츠를 추가할 때:

1. **기존 아이템과 비교**: 유사한 아이템과 통계 비교
2. **진행 확인**: 게임의 적절한 단계에 맞는지 확인
3. **플레이테스트**: 게임 내에서 실제로 테스트
4. **피드백 수집**: 다른 플레이어의 의견 수렴

## 일반적인 밸런스 문제

### 너무 강력함

- 최소한의 노력으로 너무 쉽게 획득
- 게임 메커니즘을 무효화함
- 다른 옵션을 쓸모없게 만듦

### 너무 약함

- 노력에 비해 가치가 없음
- 유사한 아이템이 모든 면에서 더 나음
- 실용적인 용도가 없음

## 도구

밸런스 체크에 도움이 되는 도구:

- **스프레드시트**: 아이템 통계 비교
- **JSON 도구**: 정의 검증
- **게임 내 디버그**: 값 테스트
