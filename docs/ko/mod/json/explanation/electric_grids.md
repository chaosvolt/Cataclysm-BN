# 전력망

전력망은 건물이나 차량 간에 전력을 분배할 수 있는 시스템입니다.

## 개요

전력망을 사용하면 다음을 수행할 수 있습니다:

- 여러 전력 소스를 단일 망에 연결
- 여러 전력 소비자를 단일 망에 연결
- 전력 소스 간 자동 부하 분산
- 배터리 저장 시스템 생성

## 작동 방식

전력망은 다음으로 구성됩니다:

- **전력 소스**: 태양광 패널, 발전기, 배터리 등
- **전력 소비자**: 조명, 가전제품, 충전소 등
- **연결**: 케이블이나 전선으로 구성요소를 연결

망은 매 턴마다 업데이트되어 사용 가능한 모든 소스에서 전력을 분배합니다.

## JSON 정의

전력망 구성요소는 가구나 지형에서 다음 플래그를 사용하여 정의됩니다:

```json
{
  "type": "furniture",
  "id": "f_solar_panel",
  "flags": ["POWER_SOURCE"],
  ...
}
```

```json
{
  "type": "furniture", 
  "id": "f_light",
  "flags": ["POWER_CONSUMER"],
  ...
}
```

## 관련 플래그

- `POWER_SOURCE` - 전력을 생성함
- `POWER_CONSUMER` - 전력을 소비함
- `POWER_STORAGE` - 전력을 저장함
- `POWER_CABLE` - 전력 전송용 케이블

## 참고사항

- 전력망은 아직 개발 중인 기능입니다
- 모든 가구/지형이 전력망을 지원하는 것은 아닙니다
- 향후 업데이트에서 구현이 변경될 수 있습니다
