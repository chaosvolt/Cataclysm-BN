# 근접 밸런스 스프레드시트

이 문서는 근접 무기 밸런스를 위한 스프레드시트 사용법을 설명합니다.

## 개요

근접 밸런스 스프레드시트는 근접 무기를 비교하고 균형을 맞추는 데 도움이 되는 도구입니다.

## 스프레드시트 접근

스프레드시트는 다음 위치에서 사용할 수 있습니다:

- Google Sheets (커뮤니티 공유)
- CSV 내보내기 (로컬 분석)

## 열

### 기본 정보

- **ID**: 아이템 ID
- **Name**: 아이템 이름
- **Type**: 무기 유형 (베기, 찌르기 등)

### 통계

- **Weight**: 무게 (g)
- **Volume**: 부피 (ml)
- **Bash**: 타격 데미지
- **Cut**: 베기 데미지
- **Pierce**: 관통 데미지
- **To-Hit**: 명중 보너스

### 계산된 값

- **DPS**: 초당 데미지
- **Speed**: 공격 속도
- **Reach**: 사정거리
- **Techniques**: 사용 가능한 기술

### 밸런스 지표

- **Balance Score**: 전반적인 밸런스 점수
- **Tier**: 무기 티어 (초반/중반/후반)
- **Rarity**: 희귀도

## 사용법

### 1. 새 무기 추가

1. 스프레드시트에 새 행 추가
2. 기본 통계 입력
3. 계산된 값 확인
4. 유사한 무기와 비교

### 2. 기존 무기 비교

1. 비교하려는 무기 찾기
2. 주요 통계 비교
3. 밸런스 점수 확인
4. 필요시 조정

### 3. 밸런스 확인

1. 동일 티어 무기 필터링
2. DPS 범위 확인
3. 이상값 식별
4. 조정 권장사항 메모

## 밸런스 공식

### DPS 계산

```
DPS = (Bash + Cut + Pierce) / Attack_Time
```

### 공격 시간

```
Attack_Time = Base_Time * (1 - To_Hit_Bonus/100) * Weight_Factor
```

### 밸런스 점수

```
Balance_Score = DPS / (Weight + Volume/100) * Rarity_Modifier
```

## 무기 티어

### 초반 (티어 1)

- **DPS**: 5-15
- **무게**: <1kg
- **예시**: 파이프, 배트, 칼

### 중반 (티어 2)

- **DPS**: 15-30
- **무게**: 1-2kg
- **예시**: 검, 도끼, 창

### 후반 (티어 3)

- **DPS**: 30-50
- **무게**: 2-4kg
- **예시**: 카타나, 전투 도끼, 할버드

### 엔드게임 (티어 4)

- **DPS**: 50+
- **무게**: 다양함
- **예시**: 모노 블레이드, 파워 무기

## 밸런스 가이드라인

### 무게 대 데미지

- 무거운 무기 = 더 높은 데미지
- 가벼운 무기 = 더 빠른 공격
- 균형: DPS는 유사해야 하지만 플레이 스타일이 다름

### 희귀도 대 파워

- 일반: 평균 능력
- 희귀: 10-20% 향상
- 매우 희귀: 20-30% 향상
- 고유: 30-50% 향상

### 특수 기능

무기에 특수 기술이나 기능이 있는 경우:

- 기본 통계를 10-20% 감소
- 유연성에 대한 균형
- 플레이 스타일 다양성 고려

## 모범 사례

1. **비교**: 항상 유사한 무기와 비교
2. **테스트**: 게임 내에서 값 테스트
3. **반복**: 필요시 조정
4. **문서화**: 밸런스 결정 문서화
5. **피드백**: 커뮤니티 의견 구하기

## 일반적인 문제

### 너무 강력함

- DPS가 티어보다 훨씬 높음
- 비용이나 희귀도가 낮음
- 단점 없음

**수정**: 데미지 감소, 무게 증가, 또는 희귀도 증가

### 너무 약함

- DPS가 티어보다 훨씬 낮음
- 다른 무기에 비해 실용성 없음
- 높은 비용과 낮은 이점

**수정**: 데미지 증가, 무게 감소, 또는 특수 기술 추가

## 도구

- 정렬 및 필터링
- 자동 계산
- 차트 및 그래프
- 데이터 검증

### 내보내기 옵션
