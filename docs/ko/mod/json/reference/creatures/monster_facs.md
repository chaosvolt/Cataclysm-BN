> [!NOTE]

> 이 문서는 최근 `JSON INFO`에서 분리되었으며 추가 작업이 필요할 수 있습니다

### 몬스터 팩션

| 식별자         | 설명                                                                                                        |
| -------------- | ----------------------------------------------------------------------------------------------------------- |
| `name`         | 고유 ID. 하나의 연속된 단어여야 하며, 필요할 때 밑줄을 사용합니다.                                          |
| `base_faction` | 선택적 기본 팩션. 다른 팩션과의 관계는 이로부터 상속되며 다른 팩션의 이 팩션에 대한 관계는 이를 확인합니다. |
| `by_mood`      | 화가 났을 때 이 팩션에 대해 적대적이고, 그렇지 않으면 중립입니다. 다른 모든 팩션에 대한 기본 태도입니다.    |
| `neutral`      | 항상 이 팩션에 대해 중립입니다.                                                                             |
| `friendly`     | 항상 이 팩션에 대해 우호적입니다. 기본적으로 팩션은 자신에게 우호적입니다.                                  |
| `hate`         | 항상 이 팩션에 대해 적대적입니다. 가능한 경우 이 팩션의 몬스터로 대상을 변경합니다.                         |

```json
{
  "name": "cult",
  "base_faction": "zombie",
  "by_mood": ["blob"],
  "neutral": ["nether"],
  "friendly": ["blob"],
  "hate": ["fungus"]
}
```
