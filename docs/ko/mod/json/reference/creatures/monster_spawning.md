title: 몬스터 생성 시스템
---

> [!NOTE]
>
> 이 문서는 최근 `JSON INFO`에서 분리되었으며 추가 작업이 필요할 수 있습니다

### 몬스터 그룹

#### 그룹 정의

| 식별자      | 설명                                                                                                                                                                                                                |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `name`      | 고유 ID. 하나의 연속된 단어여야 하며, 필요한 경우 밑줄을 사용합니다.                                                                                                                                                |
| `default`   | 기본 몬스터, 나머지 생성 확률을 자동으로 채웁니다.                                                                                                                                                                  |
| `monsters`  | 생성할 몬스터를 선택하기 위해 게임은 1000개의 항목을 만들고 하나를 선택합니다. 각 몬스터는 "freq"와 같은 수의 항목을 가지며 기본 몬스터가 나머지를 채웁니다. 단일 몬스터 정의를 만드는 방법은 아래 표를 참조하세요. |
| `is_safe`   | (bool) 안전 모드 경고를 트리거하지 않도록 확인합니다.                                                                                                                                                               |
| `is_animal` | (bool) 해당 그룹에 일반 동물만 있는지 확인합니다.                                                                                                                                                                   |

#### 몬스터 정의

| 식별자            | 설명                                                                                                                                                                                                                                                                                                                                                               |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `monster`         | 몬스터의 고유 ID, 예: `"mon_zombie"`.                                                                                                                                                                                                                                                                                                                              |
| `freq`            | 발생 확률, x/1000.                                                                                                                                                                                                                                                                                                                                                 |
| `cost_multiplier` | 제한된 수의 몬스터를 생성하는 경우 이 정의의 각 몬스터가 몇 개의 몬스터로 계산되어야 하는지.                                                                                                                                                                                                                                                                       |
| `pack_size`       | (_선택 사항_) 함께 생성되어야 하는 이 그룹의 몬스터의 최소 및 최대 수. (기본값: `[1,1]`)                                                                                                                                                                                                                                                                           |
| `conditions`      | 조건은 몬스터가 생성되는 시기를 제한합니다. 유효한 옵션: `SUMMER`, `WINTER`, `AUTUMN`, `SPRING`, `DAY`, `NIGHT`, `DUSK`, `DAWN`. 여러 시간대 조건(`DAY`, `NIGHT`, `DUSK`, `DAWN`)은 해당 조건 중 하나라도 생성을 유효하게 만들도록 결합됩니다. 여러 계절 조건(`SUMMER`, `WINTER`, `AUTUMN`, `SPRING`)은 해당 조건 중 하나라도 생성을 유효하게 만들도록 결합됩니다. |
| `starts`          | (_선택 사항_) 이 항목은 이 시간 이후에 활성화됩니다. (시간 단위로 측정)                                                                                                                                                                                                                                                                                            |
| `ends`            | (_선택 사항_) 이 항목은 이 시간 이후에 비활성화됩니다. (시간 단위로 측정)                                                                                                                                                                                                                                                                                          |

```json
{
  "name": "GROUP_ANT",
  "default": "mon_ant",
  "monsters": [
    { "monster": "mon_ant_larva", "freq": 40, "multiplier": 0 },
    { "monster": "mon_ant_soldier", "freq": 90, "multiplier": 5 },
    { "monster": "mon_ant_queen", "freq": 0, "multiplier": 0 },
    {
      "monster": "mon_thing",
      "freq": 100,
      "multiplier": 0,
      "pack_size": [3, 5],
      "conditions": ["DUSK", "DAWN", "SUMMER"]
    }
  ]
}
```
