# 새로운 돌연변이 분류

## 변이 분류(Mutation Category)

돌연변이는 다양한 분류로 구성되어 있으며, 분류를 단순화하고 새로운 돌연변이를 추가하기 위해 `MUTATION_CATEGORY` 타입이 만들어졌습니다.

## `MUTATION_CATEGORY`

```json
{
  "type": "MUTATION_CATEGORY",
  "id": "GASTROPOD",
  "name": "Gastropod",
  "threshold_mut": "THRESH_GASTROPOD",
  "mutagen_message": "당신의 발이 하나의 근육질 발로 뭉쳐지는 것을 느낍니다.",
  "iv_message": "당신은 기절하며 소금을 위해 매우 강렬하게 갈망합니다.",
  "iv_sound_message": "슬러핑 소리를 들을 수 있습니다.",
  "iv_noise": 20,
  "iv_sleep": true,
  "iv_sleep_dur": 300,
  "junkie_message": "소금 섭취에 대한 욕구를 느낍니다.",
  "memorial_message": "당신의 피부가 축축해졌습니다."
}
```

| 필드                 | 설명                                                                                                                                                                                                                 |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `type`               | 정의의 유형이며, 항상 `MUTATION_CATEGORY`입니다.                                                                                                                                                                     |
| `id`                 | 이 돌연변이 분류의 고유 식별자입니다.                                                                                                                                                                                |
| `name`               | 플레이어에게 표시될 이름입니다.                                                                                                                                                                                      |
| `threshold_mut`      | 이 돌연변이 분류의 임계 돌연변이입니다.                                                                                                                                                                              |
| `mutagen_message`    | 돌연변이 유발제를 복용할 때 표시될 메시지입니다.                                                                                                                                                                     |
| `iv_message`         | 혈청을 복용할 때 표시될 메시지입니다.                                                                                                                                                                                |
| `iv_sound_message`   | 소리를 설정하면, 혈청을 복용할 때 플레이어에게 표시될 메시지입니다.                                                                                                                                                  |
| `iv_noise`           | 혈청을 복용할 때 발생하는 소리의 크기입니다. 이것은 `iv_sound_message`와 함께 사용됩니다. `shouting`과 비슷한 값은 대략 15이며, 이 값은 좀비를 끌어들일 수 있습니다. 더 높은 값은 더 먼 좀비를 끌어들일 수 있습니다. |
| `iv_sleep`           | 혈청을 복용할 때 플레이어를 잠들게 할지 여부입니다.                                                                                                                                                                  |
| `iv_sleep_dur`       | 플레이어가 자는 시간입니다(이동 횟수로 측정됩니다, 100이동 = 1턴).                                                                                                                                                   |
| `junkie_message`     | 돌연변이 분류에 중독되었을 때 표시될 메시지입니다.                                                                                                                                                                   |
| `memorial_message`   | 임계값 돌연변이를 얻었을 때 추모문에 추가될 메시지입니다.                                                                                                                                                            |
| `vitamin`            | 필수는 아니지만, 존재하는 경우 이 분류와 관련된 비타민 타입을 정의합니다. 기본적으로 분류의 ID를 따릅니다.                                                                                                           |
| `mutagen_hunger`     | 필수는 아니지만, 존재하는 경우 돌연변이 유발제를 복용할 때 얻는 배고픔의 양을 정의합니다. 기본값은 `10`입니다.                                                                                                       |
| `mutagen_thirst`     | 필수는 아니지만, 존재하는 경우 돌연변이 유발제를 복용할 때 얻는 갈증의 양을 정의합니다. 기본값은 `10`입니다.                                                                                                         |
| `mutagen_fatigue`    | 필수는 아니지만, 존재하는 경우 돌연변이 유발제를 복용할 때 얻는 피로도의 양을 정의합니다. 기본값은 `5`입니다.                                                                                                        |
| `iv_morale`          | 필수는 아니지만, 존재하는 경우 혈청을 복용할 때 얻는 사기의 양을 정의합니다. 기본값은 `0`입니다.                                                                                                                     |
| `iv_morale_max`      | 필수는 아니지만, 존재하는 경우 혈청으로 얻을 수 있는 최대 사기를 정의합니다. 기본값은 `0`입니다.                                                                                                                     |
| `iv_sound`           | 필수는 아니지만, 존재하는 경우 혈청을 복용할 때 발생하는 소리를 정의합니다. 기본값은 `false`입니다.                                                                                                                  |
| `iv_sound_id`        | 필수는 아니지만, 존재하는 경우 혈청을 복용할 때 발생하는 소리의 ID를 정의합니다. 기본값은 `shout`입니다.                                                                                                             |
| `iv_sound_variant`   | 필수는 아니지만, 존재하는 경우 혈청을 복용할 때 발생하는 소리의 변형을 정의합니다. 기본값은 `default`입니다.                                                                                                         |
| `iv_sound_tim_typed` | 필수는 아니지만, 존재하는 경우 혈청을 복용할 때 발생하는 소리의 시간 타입을 정의합니다. 기본값은 `false`입니다.                                                                                                      |
