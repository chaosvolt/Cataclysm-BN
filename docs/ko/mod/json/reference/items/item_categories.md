# 아이템 카테고리

아이템 카테고리는 인벤토리와 제작 메뉴에서 아이템을 구성하고 그룹화하는 데 사용됩니다.

## 기본 구조

아이템 카테고리는 `item_category` 타입을 사용하여 정의됩니다:

```json
{
  "type": "item_category",
  "id": "weapons",
  "name": "무기",
  "sort_rank": 100,
  "priority_zones": ["LOOT_WEAPONS"]
}
```

## 필드

| 필드             | 타입    | 설명                                                       |
| ---------------- | ------- | ---------------------------------------------------------- |
| `type`           | string  | 반드시 `"item_category"`여야 합니다.                       |
| `id`             | string  | 이 카테고리의 고유 식별자.                                 |
| `name`           | string  | 플레이어에게 표시되는 이름.                                |
| `sort_rank`      | integer | 카테고리 정렬 순서 (낮은 숫자가 먼저).                     |
| `priority_zones` | array   | 선택 사항. 이 카테고리의 아이템에 대한 우선순위 정렬 구역. |

## 참고사항

- 카테고리는 메뉴 구성 및 자동 정렬에 사용됩니다.
- 각 아이템은 해당 정의에서 카테고리를 지정합니다.
