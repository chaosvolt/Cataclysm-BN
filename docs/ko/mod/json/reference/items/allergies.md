# 알레르기

알레르기는 플레이어가 특정 재료로 만들어진 아이템을 착용할 때 부정적인 효과가 발생하는 메커니즘입니다. 이는 특정 돌연변이나 특성과 함께 사용하여 게임플레이에 추가적인 도전과 현실감을 제공합니다.

## 기본 구조

알레르기는 돌연변이 또는 특성 정의의 `restrict_clothing_on_fire`와 같은 필드를 통해 정의됩니다:

```json
{
  "type": "mutation",
  "id": "WOOL_ALLERGY",
  "name": "양모 알레르기",
  "description": "양모에 알레르기가 있어 양모로 만든 옷을 입으면 불편함을 느낍니다.",
  "restrict_clothing_on_fire": "WOOL"
}
```

## 동작 방식

플레이어가 제한된 재료로 만들어진 아이템을 착용할 때:

1. 게임이 아이템의 재료를 확인합니다.
2. 재료가 돌연변이/특성의 제한 재료와 일치하면 부정적인 효과가 트리거됩니다.
3. 일반적인 효과는 가려움, 통증 또는 사기 감소입니다.

## 사용 예시

### 예시 1: 양모 알레르기 특성

```json
{
  "type": "trait",
  "id": "WOOL_ALLERGY",
  "name": "양모 알레르기",
  "description": "양모에 심각한 알레르기가 있습니다. 양모로 만든 옷을 입으면 피부 자극을 유발합니다.",
  "points": -1,
  "restrict_clothing_on_fire": "WOOL"
}
```

이 특성은 플레이어가 양모 아이템을 착용할 때 불이익을 제공합니다.

### 예시 2: 합성 섬유 민감성 돌연변이

```json
{
  "type": "mutation",
  "id": "SYNTHETIC_SENSITIVITY",
  "name": "합성 섬유 민감성",
  "description": "돌연변이로 인해 피부가 합성 재료에 민감해졌습니다.",
  "points": -2,
  "restrict_clothing_on_fire": "PLASTIC"
}
```

## 참고사항

- 알레르기는 특성 및 돌연변이 시스템과 결합되어 캐릭터 빌드에 균형을 맞춥니다.
- 플레이어는 알레르기가 있는 재료로 만든 옷을 피하거나 부정적인 효과를 감수해야 합니다.
- 일부 돌연변이는 특정 재료에 대한 내성을 제공할 수도 있습니다.
