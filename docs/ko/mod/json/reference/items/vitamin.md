# vitamin

## 정의

```json
{
  "id": "iron",
  "type": "vitamin",
  "vit_type": "vitamin",
  "name": "Iron",
  "excess": "hypervitaminosis",
  "deficiency": "anemia",
  "min": -12000,
  "max": 3600,
  "rate": "15 m",
  "flags": [ "FOO" ],
  "disease": [ [ -4800, -5600 ], [ -5601, -6400 ], [ -6401, -12000 ] ],
  "disease_excess": [ [ 10, 19 ], [ 20, 29 ], [ 30, 40 ] ]
},
```

### `id`

필수. 비타민의 id.

### `type`

필수. `vitamin`이어야 합니다.

### `vit_type`

비타민의 타입. 유효한 값은:

#### `vitamin`

간소화된 영양 시스템이 활성화되면, 이 비타민은 어떤 아이템에도 추가되지 않으며 게임이 플레이어로부터
이를 가져오려고 할 때 0을 반환합니다. 영양 관련 비타민만 이 타입을 가져야 합니다.

#### `toxin`

이것은 독성 화학물질 또는 성분입니다. 현재 효과가 없습니다.

#### `drug`

이것은 약물입니다. 현재 효과가 없습니다.

#### `counter`

이것은 독소, 비타민, 약물이 아닌 무언가의 카운터입니다. 현재 효과가 없습니다.

### `name`

아이템 메뉴의 비타민 표시와 같이 비타민이 표시되는 곳에서 나타나는 이름.

### `deficiency`

이 비타민의 결핍으로 인해 발동되는 효과의 id.

### `excess`

이 비타민의 과잉으로 인해 발동되는 효과의 id.

### `min`

플레이어가 가질 수 있는 이 비타민의 최소량.

### `max`

플레이어가 가질 수 있는 이 비타민의 최대량.

### `rate`

이 비타민 한 단위를 잃는 데 걸리는 시간.

### `flags`

문자열 플래그의 배열, 유효한 플래그는 아래의 플래그 섹션을 참조하세요.

### `disease`

이 비타민 결핍의 임계값. 목록의 각 쌍은 해당 결핍 단계의 시작점과 끝점을 결정합니다.
각 결핍 단계는 `deficiency`에 정의된 효과의 강도 레벨에 해당합니다.

### `disease_excess`

이 비타민 과잉의 임계값. 목록의 각 쌍은 해당 과잉 단계의 시작점과 끝점을 결정합니다.
각 과잉 단계는 `excess`에 정의된 효과의 강도 레벨에 해당합니다.

## 플래그

- `NO_DISPLAY` - 이 비타민은 음식을 조사할 때 표시되지 않습니다
