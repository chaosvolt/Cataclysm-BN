# JSON 플래그

## 예

```json
{
  "type": "json_flag", // 필수 유형
  "id": "GENERIC_FLAG", // 플래그 ID
  "context": [], // 아무것도 하지 않지만 반드시 존재해야 하는 Fluff 필드
  "craft_inherit": true, // 이 플래그로 만든 아이템은 이 플래그를 유지합니다.
  "requires_flag": true, // 차량 부품 플래그에 사용되며 타일에 이 ID가 있는 다른 부품이 필요합니다.
  "inherit": true, // 아이템 모드는 이 플래그를 아이템에 전달합니다.
  "tag": "string" // 항목에 이 플래그가 있는 경우 항목의 UI 표시 이름에 추가되는 번역 가능한 문자열
}
```

-

## 메모

- 일부 플래그(아이템, 효과, 차량 부품)는 `flags.json` 또는 `vp_flags.json`에 정의되어야 합니다.
  (유형: `json_flag`)이 올바르게 작동합니다.
- 하나의 카테고리나 아이템 유형을 위한 많은 플래그가 다른 카테고리나 아이템에도 사용될 수 있습니다.
  유형. else 플래그를 사용할 수 있는 곳을 실험해 보세요.
- 공격 및 방어 깃발은 착용 가능한 모든 아이템 유형에 사용할 수 있습니다.

## 상속

아이템을 제작할 때 해당 아이템을 제작하는 데 사용된 구성 요소로부터 플래그를 상속받을 수 있습니다. 이
상속될 플래그에 `"craft_inherit": true` 항목이 있어야 합니다. 당신이 원하지 않는 경우
특정 항목을 제작할 때 플래그를 상속하려면 다음의 배열인 delete_flags 멤버를 지정하세요.
문자열. 여기에 지정된 플래그는 제작 시 결과 아이템에서 제거됩니다. 이것은
플래그 상속을 재정의하지만 항목 유형 자체의 일부인 플래그는 삭제하지 않습니다.

### 플래그

`WATCH` 및 `ALARMCLOCK`과 같은 일부 갑옷 플래그는 다른 항목 유형과 호환됩니다. 실험
어떤 플래그가 다른 곳에서 작동하는지 찾으려면

- `ACTIVE_CLOAKING` 활성화된 동안 UPS을 소모하여 투명화를 제공합니다.
- `ALARMCLOCK` 알람시계 기능이 있습니다.
- `ALLOWS_NATURAL_ATTACKS` 자연 공격이나 돌연변이로 인한 유사한 이점을 방지하지 않습니다.
  손가락 끝 면도기 등 관련 신체 부위를 덮는 대부분의 품목과 같습니다.
- `AURA` 이 아이템은 형이상학적인 효과를 위해 의도된 외부 오라 레이어에 들어갑니다.
- `BAROMETER` 이 장비에는 정확한 기압계(대기를 측정하는 데 사용됨)가 장착되어 있습니다.
  압력).
- `BELTED` 배낭과 겉옷 위에 착용하는 물건을 위한 레이어입니다.
- `BLIND` 착용하는 동안 착용자의 눈을 멀게 하고 섬광탄에 대한 명목상의 보호를 제공합니다.
- `BLOCK_WHILE_WORN` 착용한 갑옷이나 방패를 사용하여 공격을 차단할 수 있습니다. 또한 참조하십시오
  `Techniques` 섹션.
- `BULLET_IMMNUE` 이 깃발이 달린 아이템을 착용하면 총알 피해에 면역이 됩니다.
- `CLIMATE_CONTROL` 이 의류에는 일종의 온도 조절 장치가 있어 몸을 따뜻하게 유지하거나
  주변 온도와 체온에 따라 더 시원해집니다.
- `COLLAR` 이 옷은 칼라가 넓어 입을 따뜻하게 유지할 수 있습니다.
- `DEAF` 플레이어를 귀머거리로 만듭니다.
- `ELECTRIC_IMMUNE` 이 장비는 방전으로부터 사용자를 완벽하게 보호합니다.
- `FANCY` 플레이어가 옷을 입지 않은 경우 이 옷을 입으면 사기 보너스를 받습니다.
  `Fashion Deficient` 특성.
- `FIX_FARSIGHT` 이 장비는 원시를 교정합니다.
- `FIX_NEARSIGHT` 근시를 교정해주는 장비입니다.
- `FLOTATION` 플레이어가 깊은 물에 빠지는 것을 방지합니다. 또한 수중 다이빙을 방지합니다.
- `FRAGILE` 이 장비는 일반 장비보다 손상에 대한 저항력이 약합니다.
- `HELMET_COMPAT` SKINTIGHT 또는 OVERSIZE가 아니지만 헬멧과 함께 착용할 수 있는 아이템입니다.
- `HOOD` 추가적인 보온이나 물 공급을 위해 이 옷이 조건에 따라 머리를 덮도록 허용합니다.
  보호. 플레이어의 머리가 방해받지 않는 경우
- `HYGROMETER` 이 장비에는 정확한 습도계가 장착되어 있습니다(온도를 측정하는 데 사용됨).
  습도).
- `NO_TAKEOFF` 해당 플래그가 붙은 아이템은 떼어낼 수 없습니다.
- `NO_QUICKDRAW` 발사 키를 누른 상태에서 이 권총집에서 아이템을 꺼내겠다고 제안하지 마세요.
  플레이어의 손이 비어 있습니다
- `ONLY_ONE` 하나만 착용할 수 있습니다.
- `OUTER` 겉옷 레이어.
- `OVERSIZE` 부담/돌연변이/생체공학/등에 관계없이 항상 착용할 수 있지만 다른 것은 방지합니다.
  그 위에 입는 옷.
- `PARTIAL_DEAF` 소리의 볼륨을 안전한 수준으로 줄입니다.
- `PERSONAL` 이 아이템은 형이상학적인 효과를 위해 고안된 개인 오라 레이어에 사용됩니다.
- `POCKETS` 플레이어의 손이 차갑고 플레이어가 휘두르는 경우 손의 따뜻함이 증가합니다.
  아무것도.
- `POWERARMOR_EXO` 아이템을 파워 아머의 주요 외골격으로 표시합니다.
- `POWERARMOR_EXTERNAL` 항목을 신체 부위와 외골격을 덮는 외부 조각으로 표시합니다.
  그렇지 않습니다.
- `POWERARMOR_MOD` 아이템을 외골격/외부 장치에 착용하는 파워 아머 모드로 표시합니다.
  조각.
- `POWERARMOR_COMPATIBLE` 다른 매개변수에도 불구하고 아이템이 파워 아머와 호환되도록 만듭니다.
  실패.
- 착용 시 `PSYSHIELD_PARTIAL` 25% 확률로 공포_마비 몬스터의 공격으로부터 보호됩니다.
- `RAD_PROOF` 이 옷은 방사선으로부터 당신을 완벽하게 보호합니다.
- `RAD_RESIST` 이 옷은 방사선으로부터 부분적으로 보호해 줍니다.
- `RAINPROOF` 덮은 신체 부위가 비에 젖는 것을 방지합니다.
- `REQUIRES_BALANCE` 안정적으로 사용하려면 일정한 균형이 필요한 장비입니다. 플레이어가 타격을 받은 경우
  착용하는 동안 쓰러질 가능성이 있습니다.
- `RESTRICT_HANDS` 플레이어가 양손으로 무기를 휘두르는 것을 방지하고 다음과 같은 경우 한손으로 사용하도록 강제합니다.
  무기가 그것을 허용합니다.
- `ROLLER_ONE` ROLLER_QUAD의 덜 안정적이고 느린 버전이지만 여전히 플레이어가 이동할 수 있습니다.
  걷는 속도보다 빠르다.
- `ROLLER_QUAD`ROLLER_INLINE과 ROLLER_ONE 사이의 중간 선택이지만 더 안정적이며
  더 빠르게 움직일수록 ROLLER_ONE보다 평탄하지 않은 지형에 대한 페널티가 더 가혹해집니다.
- `ROLLER_INLINE` 더 빠르지만 전체적으로 안정성이 떨어지므로 평평하지 않은 지형에 대한 페널티는 더욱 가혹합니다.
- `SEMITANGIBLE` 착용 시 아이템이 부담 시스템에 참여하는 것을 방지합니다.
- `COMPACT` 착용 시 아이템이 레이어링 부담 시스템에 참여하는 것을 방지합니다.
- `SKINTIGHT` 속옷 레이어.
- `SLOWS_MOVEMENT` 이 옷은 이동 비용에 1.1을 곱합니다.
- `SLOWS_THIRST` 이 옷은 플레이어의 목마름 속도를 다음과 같이 증가시킵니다.
  0.70.
- `STURDY` 이 옷은 일반 옷보다 손상에 대한 저항력이 훨씬 더 강합니다.
- `SUN_GLASSES` 햇빛 아래 있을 때 눈부심을 방지합니다.
- `SUPER_FANCY` 플레이어가 `FANCY`에 비해 추가 도덕적 보너스를 제공합니다.
  `Fashion Deficient` 특성.
- `SWIM_GOGGLES` 물속에서 훨씬 더 멀리 볼 수 있습니다.
- `THERMOMETER` 이 장비에는 정확한 온도계가 장착되어 있습니다(측정하는 데 사용됨).
  온도).
- `VARSIZE` 재봉을 통해 꼭 맞게 제작할 수 있습니다.
- `WAIST` 허리에 착용하는 기타 벨트용 레이어입니다.
- `WATCH` 시계 역할을 하며 플레이어가 실제 시간을 볼 수 있도록 합니다.
- `WATERPROOF` 어떤 상황에서도 가려진 신체 부위가 젖는 것을 방지합니다.
- `WATER_FRIENDLY` 아이템의 신체 부위가 물에 비친화적인 것으로 간주되는 것을 방지합니다.

  ```
  -X- -Y-
   |      오
  ```

## 돌연변이

#### 돌연변이 플래그

Mutation 플래그는 다른 플래그와 다른 JSON 유형을 사용합니다. json/flags_mutation.json을 참조하세요. 기본
차이점은 `conflicts` 및 `requires`가 추가할 수 있는 유일한 추가 속성이라는 것입니다.
그들에게.

다음은 현재 게임 코드에서 사용되는 모든 특성 플래그를 보여줍니다. 특성 플래그도 다음과 같아야 합니다.
NPC 대화 조건에서 사용하려면 JSON에서 정의해야 합니다.

- `CANNIBAL` 인간의 시체를 도살해도 사기 저하가 없으며, 책의 부정적인 사기를 전환합니다.
  BOOK_CANNIBAL 플래그를 양수로 설정하면 인간 고기에 대한 경고를 건너뜁니다. NOTE: 건너뛰기만 합니다.
  경고에 따르면, 인간 육체를 먹는 실제 사기 효과는 여전히 관련 중 하나를 필요로 합니다.
  특성. 이 깃발이 있는 사용자 지정 특성은 경고를 건너뛰고 사기 저하를 겪습니다.
- `NEED_ACTIVE_TO_MELEE` 이 플래그가 있는 돌연변이는 비무장 보너스만 제공합니다.
  켜졌습니다.

  돌연변이 임계값을 위반할 수 있는 능력)에 속하는 것으로 간주되는 모든 카테고리에 적용됩니다.

### WBLOCK_X

다음 무기 기술에는 몇 가지 추가 용도가 있습니다. 방어하는 기술들입니다
아이템이 근접 공격을 차단하는 데 도움이 되도록 허용하고 몇 가지 추가 특수 용도를 사용합니다.

- `WBLOCK_1` "Medium blocking ability"
- `WBLOCK_2` "High blocking ability"
- `WBLOCK_3` "Very high blocking ability"

이러한 기술 중 하나를 사용하는 아이템을 사용하여 차단으로 감소된 피해에 보너스를 제공할 수 있습니다.
비교하거나 `BLOCK_WHILE_WORN` 플래그가 있는 갑옷도 이 보너스를 사용할 수 있습니다.
아이템을 착용하고 방패 역할을 합니다. 추가적으로, 조합된 아이템을 휘두르거나 착용하는 행위
이러한 기술 중 하나와 해당 플래그를 사용하면 아이템이 신체를 겨냥한 발사체를 차단할 수 있습니다.
품목이 덮지 않는 부분(또는 다음 요건을 충족하는 휘두르는 품목의 경우 덮지 않는 부분)
그 전제 조건). 이런 일이 발생할 가능성은 `coverage` 비율에 따라 결정됩니다.
일반 방어구 값으로 사용되는 아이템으로, 어떤 차단 기술에 따라 페널티가 감소합니다.
그것은 소유하고 있습니다. 다리를 타격하는 탄환을 차단할 가능성이 있습니다. (다시 말하지만, 갑옷이
이미 기본적으로 다리를 덮도록 설정되어 있으며, 이 경우 `coverage`를 정상적으로 사용합니다.)
처벌을 받았습니다. (JSON 작성자가 고안한 이유가 무엇이든,
역장 아이템(예: 역장 아이템) 아이템은 이미 갑옷으로 발을 덮고 있는 방패입니다.

### 기타 차량 부품이 필요한 차량 부품

다른 자동차 부품에 대한 요구 사항은 json 플래그에 대해 `requires_flag`을 설정하여 정의됩니다.
깃발. `requires_flag`은 이 플래그가 있는 부분에 필요한 다른 플래그입니다.
