# 세계 유형

월드 타입은 차원에 대한 데이터를 저장하는 데 사용됩니다. 기본 포함

> [!WARNING]
> 치수는 아직 다소 실험적이므로 사용에 따른 책임은 사용자 본인에게 있습니다.

### 필드

| 식별자               | 설명                                                                              |
| -------------------- | --------------------------------------------------------------------------------- |
| id                   | (_필수_) 고유 ID입니다.                                                           |
| name                 | (_필수_) 게임 내 이름이 표시됩니다.                                               |
| description          | (_선택 사항_) 차원에 대한 설명                                                    |
| region_settings      | (_선택 사항_) 기본값은 `default`, 사용할 지역 설정의 ID입니다.                    |
| generate_overmap     | (_선택 사항_) 기본값은 true입니다. 호수, 숲, 늪 연결이 생성되어야 합니다.         |
| infinite_bounds      | (_선택적_) 기본값은 true입니다. 기본 차원처럼 경계가 없는 차원입니다.             |
| boundary_terrain     | (_선택 사항_) 기본값 "boundary_terrain", 차원 경계 주위에 배치할 지형입니다.      |
| save_prefix          | (_선택적_) 접두사로 저장할 문자열입니다.                                          |
| allow_npc_travel     | (_선택 사항_) 기본 거짓, NPC가 차원으로 이동할 수 있는 날씨입니다.                |
| allow_vehicle_travel | (_선택 사항_) 기본값은 false이며, 차량이 해당 차원으로 이동할 수 있는 날씨입니다. |
| scale_num            | (_선택 사항_) 이 차원의 여행 규모입니다. 2는 2배 더 긴 여행을 의미합니다.         |
| scale_den            | (_선택 사항_) 이 차원의 여행 규모입니다. 2는 2배 더 짧은 이동 거리를 의미합니다.  |
| sunrise_summer       | (_선택적_) 기본값 -1(게임 옵션 사용), 여름 일출 시간                              |
| sunset_summer        | (_선택적_) 기본값 -1(게임 옵션 사용), 여름 일몰 시간                              |
| sunset_winter        | (_선택적_) 기본값 -1(게임 옵션 사용), 겨울 일몰 시간                              |
| sunset_winter        | (_선택적_) 기본값 -1(게임 옵션 사용), 겨울 일몰 시간                              |
| sunrise_equinox      | (_선택 사항_) 기본값 -1(게임 옵션 사용), 춘분의 일출 시간                         |
| sunset_equinox       | (_선택 사항_) 기본값 -1(게임 옵션 사용), 춘분의 일몰 시간                         |
| pernament_daylight   | (_선택 사항_) 기본값 -1(게임 옵션 사용), 날씨를 항상 낮으로 설정합니다.           |
| pernament_night      | (_선택적_) 기본값 -1(게임 옵션 사용), 날씨를 항상 밤으로 설정합니다.              |
